# Claude Code Marketing Skills

Marketing skills for Claude Code, built from Alex Hormozi's *100 Million Dollar Leads*.

Each skill is a `SKILL.md` file that Claude loads automatically when you ask about that topic. Drop any folder into your Claude Code project and it works immediately.

---

## Skills

### [`lead-magnets/`](lead-magnets/SKILL.md)
Hormozi's 7-step lead magnet framework. Helps you create free offers that build trust and make your core offer obvious. Covers problem selection, delivery formats, naming, CTAs, and common mistakes.

---

## How to use

1. Copy the skill folder into your project
2. Claude will load it automatically when relevant
3. Or reference it directly: *"use the lead-magnets skill to help me build a free offer"*
